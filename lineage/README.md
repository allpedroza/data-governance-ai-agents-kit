## Summary
Agent specialized in identifying, analyzing, and visualizing data lineage in complex pipelines.

## Key Features
- **Multi-format support**: Python, SQL, Terraform, Databricks, Spark.
- **Impact analysis** and version comparison.
- **Interactive visuals**: Force, Hierarchical, Sankey, 3D, Radial.
- **Cycle detection and complexity metrics**.
- **Export** to JSON, Markdown, and HTML.

## Installation
```bash
pip install -r requirements.txt
```

Optional: set `OPENAI_API_KEY` and `DATA_LINEAGE_LLM_MODEL` for LLM-assisted contextual extraction.

## Quickstart
```bash
python lineage_system.py /caminho/do/projeto --visualize dashboard --report
```

The output includes interactive graphs, impact analysis, and autogenerated documentation.
